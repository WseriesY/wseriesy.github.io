语言学习与人造语言

Language Learning and Constructed Languages (Conlangs)

概况 Overview

本人的语言学习成果包括自然语言及人造语言。
已学习的自然语言中有现今仍在使用的语言，如拉达克语；也有不再广泛使用的语言，如苏美尔语、于阗语等。已学习的人造语言中包含了国际辅助语，如道本语；以及一些艺术性人造语言。
本人还自己创作了一些人造语言。我创造的一般是以纪念喜欢的人物形象为目的，在语系层面少量参考了现实原型制作的“近架空语言”，如千织语。

My language learning works contain natural languages and conlangs.
The natural languages I've learnt include some languages that are still in use, like Ladakhi, and some languages that are no longer used widely, like Sumerian, Khotanese, etc.The conlangs I've learnt include "International auxiliary languages" like Toki Pona, and some artistic conlangs.
Moreover, I've created some handmade conlangs. What I create are generally "penconlangs", like Chiorian, for the purpose of commemorating the figures I like, and that have consulted some language families in the real world.

本人自制语言概况 the Overview of My Handmade Conlangs

自第一篇自制人造语言“千织语”初步建成以来，我逐渐积累了创作经验。
目前，我的自制造语计划已涵盖了数个游戏的架空世界，其中成体系的主要是“翁法罗斯语族”。
尚在计划中的还有“弁才天国语族”、天环语、柯莱语，以及新艾利都的一些语言。

翁法罗斯语族 the Constructed Amphorian Language Family

“翁法罗斯语族”是JoshuaWillow以游戏《崩坏·星穹铁道》的地区“翁法罗斯”为取材对象设计的人造语族。
“翁法罗斯语族”下面所有造语的词源均是现实的希腊语族语言，包括但不限于迈锡尼希腊语、阿提卡古希腊语、通用希腊语乃至语言学模拟的原始希腊语等。音系和语法会参考其他语言进行设计。
我目前创作了以下4个语支：缇里希庇俄丝语支(Tribiosic)、玻吕茜亚语支(Polyxian)、塞壬-许珀耳语支(Sireno-Hyperboric)和昔涟语支(Kyrenic)。

弁才天国语族 the Constructed Benzaitengokan Language Family

“弁才天国语族”是JoshuaWillow以游戏《崩坏·星穹铁道》的地区“弁才天国”为取材对象设计的人造语族。
“弁才天国语族”下面所有造语的词源均是现实的日琉语系语言，主要是东部古日语。音系和语法会参考其他语言进行设计。
由于语言种类较少，弁才天国语族目前不设“语支-语组”这样的印欧语系属分类系统。可能仅设“语组”。目前在规划中的语言仅有“无量塔语”和“绯英语”。
